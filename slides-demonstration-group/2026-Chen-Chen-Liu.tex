\section{Problem \& Motivation}

\begin{frame}{Problem Statement}
    \begin{itemize}
        \item \textbf{What problem are you solving?} \\
        Prolonged focus during software development causes fatigue, resulting in reduced code quality.

        \vspace{0.5em}
        \item \textbf{Why does it matter?} \\
        Fatigue-induced tunnel vision leads to unnecessary variables, inefficient code structure, increased memory usage, and higher time complexity.
        
        \vspace{0.5em}
        \item \textbf{Who are the stakeholders?} \\
        Software engineers and developers, students in intensive coding courses.
    \end{itemize}
